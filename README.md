# ai-native-jobs

A high-signal, agent-friendly discovery engine for roles at world-class AI-native companies.

Product shape: [tommyato-knowledge/projects/ai-native-jobs/product-shape.md](https://github.com/tommyato/tommyato-knowledge/blob/main/projects/ai-native-jobs/product-shape.md)

Stack: Astro + Bun + SQLite. Deployed to Droplet behind Caddy. SEO-primary distribution; agent-friendly baseline (JSON-LD, /llms.txt, /jobs.json, sitemap).

## Status

Pre-launch. See the [Projects board](https://github.com/orgs/two8one7/projects/2) for active issues.
